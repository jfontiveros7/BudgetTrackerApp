# Demo Recording Checklist

## Objective
Record a clean one-take product demo in 10 to 15 minutes.

## Pre-record setup
1. Start local app and confirm homepage loads.
2. Confirm demo login works.
3. Use browser zoom at 100 percent.
4. Close unrelated tabs and notifications.
5. Use a clean desktop background.
6. Prepare one short transaction to add during recording.

## Demo account for recording
- Email: demo@konticodelabs.com
- Password: demo1234

## Shot list in exact order
1. Homepage
- Show headline and positioning for 3 to 5 seconds.
- Optional quick scroll to pricing section.

2. Login
- Navigate to login.
- Enter demo credentials.
- Submit and pause on dashboard load.

3. Dashboard overview
- Show total income, expenses, and net balance.
- Show alerts module and coach recommendation.

4. Transactions table
- Use search with a simple term like groceries.
- Change type filter to Expense and then back to All.

5. Add transaction
- Open Add Transaction.
- Add one expense entry.
- Save and return to dashboard.

6. Settings
- Open Settings.
- Show alert preferences or AI settings.
- Return to dashboard.

7. Close frame
- Final dashboard view.
- Display short CTA text overlay.

## Suggested recording settings
- Resolution: 1920 x 1080
- Frame rate: 30 fps
- Cursor highlight: on
- Mic: off for silent version, on for narrated version

## Export versions to create
1. Marketplace short demo
- Length: 60 to 90 seconds
- Best for listing conversion

2. Full walkthrough
- Length: 3 to 5 minutes
- Best for buyer trust and support reduction

## Quality checklist before publishing
1. No loading errors visible.
2. No sensitive files or credentials shown.
3. Text is readable on mobile-size previews.
4. Start and end frames look polished.
5. Video filename is marketplace-friendly.
